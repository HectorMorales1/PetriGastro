# SPEC.md - Landing Chef

## Proyecto
Landing page para chef profesional con función principal de **reserva de comida para llevar**.
Enfoque SEO máximo.

## Stack
- HTML5 semántico
- CSS3 (Flexbox + Grid)
- JavaScript Vanilla
- Sin frameworks

## Objetivos SEO
1. Posicionar #1 en Google para keywords de chef
2. Optimizar Core Web Vitals
3. Schema.org para Local SEO y Food Establishment

## Datos del Chef (Demo)
- **Nombre**: Chef Antonio Martínez
- **Especialidad**: Cocina española moderna
- **Ubicación**: Madrid, España
- **Negocio**: Comida para llevar premium

## Secciones Requeridas

### 1. Hero
- Título: "Comida gourmet para llevar en Madrid"
- Subtítulo con keyword secundario
- CTA principal: "Reservar Ahora"
- Imagen del chef trabajando

### 2. Cómo Funciona (3 pasos)
- 1. Elige tu menú
- 2. Reserva online
- 3. Recoge en tienda

### 3. Menú / Platos Destacados
- 6 platos representativos
- Imagen + nombre + descripción + precio
- Categorías: Entrantes, Principales, Postres
- Botón "Añadir a reserva"

### 4. Sobre el Chef
- Biografía breve (E-E-A-T)
- Experiencia profesional
- Foto del chef

### 5. Reservas (Sección Principal)
- Formulario de reserva:
  - Fecha
  - Hora
  - Número de personas
  - Menú seleccionado
  - Datos de contacto
- Calendario visual

### 6. Testimonios
- 3-4 reseñas de clientes
- Calificación con estrellas
- Fotos (opcional)

### 7. Contacto
- Teléfono, email, dirección
- Horarios de recogida
- Mapa embed

### 8. Footer
- Links rápidos
- Redes sociales
- Copyright
