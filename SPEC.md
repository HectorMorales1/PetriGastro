# SPEC.md - Landing Chef

## Proyecto
Landing page para chef profesional con enfoque SEO máximo.

## Stack
- HTML5 semántico
- CSS3 (Flexbox + Grid)
- JavaScript Vanilla
- Sin frameworks

## Objetivos SEO
1. Posicionar #1 en Google para keywords de chef
2. Optimizar Core Web Vitals
3. Schema.org para Local SEO

## Secciones Requeridas

### 1. Hero
- Nombre del chef + especialidad
- Keyword principal en h1
- Foto profesional del chef
- Llamada a la acción (CTA)

### 2. Sobre Mí
- Biografía con E-E-A-T
- Experiencia, formación, premios
- Foto trabajando

### 3. Menú / Platos Destacados
- 4-6 platos representativos
- Imagen + nombre + descripción
- Categorías (entrantes, principales, postres)

### 4. Testimonios
- Reseñas de clientes
- Calificación (Schema)
- Fotos de clientes (opcional)

### 5. Contacto
- Formulario de contacto
- Teléfono, email, dirección
- Mapa (Google Maps embed)
- Horarios

### 6. Footer
- Links rápidos
- Redes sociales
- Copyright

## Keywords Objetivo (Ejemplo)
- "chef [especialidad] [ciudad]"
- "restaurante [tipo] [ciudad]"
- "cocina [especialidad] terbaik [barrio]"

## Tech Specs
- Responsive: Mobile-first
- Imágenes: WebP con lazy loading
- Velocidad objetivo: LCP < 2.5s
- CLS objetivo: < 0.1
